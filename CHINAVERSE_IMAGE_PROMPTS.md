## GLOBAL STYLE GUIDE
> Add this to the END of every prompt for consistent results:

```
--style: editorial photography, cultural magazine, soft natural lighting,
rich colors, cinematic composition, high detail, 16:9 ratio,
no text overlays, no watermarks, photorealistic
```

**For Midjourney specifically, append:**
```
--ar 16:9 --style raw --q 2
```
